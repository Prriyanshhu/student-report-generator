# Student Report Generator

This Python program:
- Takes student names and their marks
- Calculates total, average, and grade
- Writes a detailed report to a text file

## Concepts Used
- Dictionaries
- Functions
- Loops
- Conditional statements
- File handling

## How to Run
1. Run the Python file
2. Enter student details as prompted
3. Check `student_report.txt` for output
